CMD behind a proxy
==================

I am using the `cmder`_, just run:

.. code-block:: coonsole

    $ set http_proxy=http://127.0.0.1:1009
    $ set https_proxy=http://127.0.0.1:1009

If u wanna know what ip address u are using:

.. code-block:: console

    $ curl https://api.myip.com/


.. _cmder: http://cmder.net/
