virtualenvwrapper-win
=====================

Install
-------
::

    pip3 install virtualenvwrapper-win

Enviroment Vars
---------------

set enviroment ``WORKON_HOME`` as ``D:\path\to\envs``

create venv
-----------
::

    mkvirtualenv venv-name

enter venv
----------
::

    workon venv-name

exit venv
---------
::

    deactivate
